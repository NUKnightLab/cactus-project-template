Knight Lab Starter Template

This is a simple starter template, which can be used in conjunction with Cactus and Blueline to rapidly create static websites.

Cactus

Cactus is a simple but powerful static website generator using Python and the Django template system. Cactus also makes it easy to develop locally and deploy your site to S3 directly. It works great for company, portfolio, personal, support websites and blogs.

Blueline

A Twitter Bootstrap-compatible framework that quickly applies the Knight Lab brand and styles to any website.

Read more about Blueline

Bootstrap

A sleek, intuitive, and powerful front-end framework for faster and easier web development, created by Twitter. Since Blueline is Bootstrap-compatible, feel free to use any Bootstrap components, and they will be automatically styled by Blueline.
